### About Me

저는 소프트웨어 엔지니어이자한 아이의 아빠입니다. 저는 한국 출신입니다. 웹개발자로 경력의 대부분을 쌓았으며 경력에 많은 부분을 프론트엔드 개발과 오픈소스에 빚지고 있습니다.

저를 대표하는 직업은 프론트엔드 개발 이지만, 경력 초반에는 서버 개발자로 경력을 시작했습니다. 

초반에는 이것저것 다양한 언어에 관심이 많아서 PHP + CodeIgniter, Python + Django, Java + Play Framework, Java + Spring Framework 등을 얕게 경험해 봤으며, 이후 JavaScript의 매력에 빠져 프론트엔드 개발자로 전향했습니다.

그후 [TOAST UI](https://ui.toast.com/)라는 오픈소스를 만드는 팀에서 활동하습니다. 퇴사 후에도  오픈소스 개발에 재미를 붙여 [Lithent](https://github.com/superlucky84/lithent), [State-ref](https://github.com/superlucky84/state-ref) 를 만들고 있으며, 특히 State-ref는 [Webpack](https://webpack.js.org/)의 초창기 핵심 개발자인 [Juho Vepsäläinen](https://survivejs.com/about-me/)가 직접 [인터뷰](https://survivejs.com/blog/state-ref-interview/)내용을 자신에 블로그에 포스팅 해주었습니다.

현재는 [중앙일보](https://www.joongang.co.kr/) 프론트엔드 개발팀에 속해 있습니다. 더 자세한 경력은 [LinkedIn](https://www.linkedin.com/in/superlucky84/) 에서 확인할수 있습니다.
